# 什么是 DTO

+ PO - Persistant Object持久对象 (bean、entity)
+ BO - Business Object (service、manager、business等)
+ VO - Value Object值对象 (from)
+ DTO - Data Transfer Object数据传输对象
+ DAO - Data Access Object数据访问对象
